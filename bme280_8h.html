esp_err_t bme280_set_oversamp ( bme280_oversampling_t  oversamp_pressure,

)

Function to set oversampling settings.

Parameters
oversamp_pressurePressure oversampling setting
oversamp_temperatureTemperature oversampling setting
oversamp_humidityHumidity oversampling setting

This function sets the oversampling settings for the BME280 sensor.

Note
The oversampling settings are as follows:
0 - one measurement
1 - oversampling x1
2 - oversampling x2
3 - oversampling x4
4 - oversampling x8
5 - oversampling x16
Returns
ESP_OK if successful, ESP_FAIL if unsuccessful

0 - 1 ms
1 - 63 ms
2 - 125 ms
3 - 250 ms
4 - 500 ms
5 - 1000 ms
6 - 10 ms
7 - 20 ms
Parameters
filter_coeffFilter coefficient setting
Note
Filter coefficient:
0 - filter off
1 - coefficient 2
2 - coefficient 4
3 - coefficient 8
4 - coefficient 16
Parameters
power_modePower mode setting
Note
Power mode:
0 - sleep mode
1 - forced mode
2 - normal mode

This function sets the standby time, filter coefficient and power mode for the BME280 sensor.
