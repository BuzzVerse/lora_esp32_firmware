void low_power_mode_enter_deep_sleep ( void  )

Function for making the device enter into the low power mode.

This function makes the device enter into the low power mode. Before entering into the low power mode, make sure to set the sleep time using the low_power_mode_set_sleep_time function.

Warning
All data in the RAM will be lost when the device enters into the low power mode.
