esp_err_t bq27441_init ( sensor_context_t *  ctx)

Initialize the BQ27441 fuel gauge.

This function unseals the device, enters configuration mode, sets the design capacity, sets the terminate voltage, and then exits configuration mode.

Parameters
ctxSensor context containing configuration data.
Returns
  • ESP_OK on success.
  • An error code on failure.
